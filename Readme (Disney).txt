Pasos para arrancar el servidor: 
 
1)Npm i para bajar las dependencias => Carpeta Node_Modules.
2)Ejecutar el comando npm run migrations => Crea la BD y las tablas con los valores por default.
3)Arranque el servidor => Nodemon app o npm start.
4)Registrate => username, email y password son los campos a solicitar.
5)Logueate con el usuario creado y copia el token recibido en los headers para probar los endpoints  => Key:usuario-token   value:"token recibido".
